toxtree.plugins.ames.rules.SA18_gen
  1. Find essential rings through SSSRFinder.findEssentialRings. This set of rings is uniquely defined.
  2. Split into set of fused rings.
  3. For each set of fused rings verify for each ring if it is composed of aromatic atoms only.If yes, the ring is considered aromatic and the counter for aromatic rings within this set of fused rings is incremented.
  4. If the ring is aromatic, it is verified if it contains an heteroaromatic atoms, and if yes, the counter for heteroaromatic rings is incremented.
  5. If the set of fused rings contains > 2 aromatic rings and no heteroaromatic rings, the rule returns true.

  1. Find essential rings through SSSRFinder.findEssentialRings. This set of rings is uniquely defined.

  3. For each set of fused rings verify for each ring if it is composed of aromatic atoms only.If yes, the ring is considered aromatic and the counter for aromatic rings within this set of fused rings is incremented.
